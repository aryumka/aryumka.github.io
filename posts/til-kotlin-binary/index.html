[TIL] 코틀린 이진수 변환

코틀린에서 이진수 변환을 하는 법

fun main() {
    val decimal = 4
    println(decimal.toString(2)) //100
}

진수를 인자로 넘겨주면 쉽게 변환이 가능하다. 이진변환 반복하기를 풀면서 역시 반복된 연산 시 탈출조건을 찾는게 핵심이라는 걸 배웠다.
